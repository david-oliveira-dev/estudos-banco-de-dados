# Módulo 04 — SQL Básico

SQL (Structured Query Language) é a linguagem padrão para interagir com bancos de dados relacionais.

## Conteúdo

| Arquivo | Tema |
|---|---|
| [select.md](select.md) | SELECT, WHERE, ORDER BY, LIMIT |
| [joins.md](joins.md) | INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN |
| [group-by.md](group-by.md) | GROUP BY, HAVING, funções de agregação |
| [insert-update-delete.md](insert-update-delete.md) | INSERT, UPDATE, DELETE |
| [scripts/schema-loja.sql](scripts/criacao-tabelas-exemplo.sql) | Banco de exemplo para praticar |

## Para praticar

1. Crie o banco com o script em `scripts/criacao-tabelas-exemplo.sql`.
2. Leia o `.md` do tópico que quer estudar.
3. Tente os exercícios em `../05-exercicios/` antes de ver o gabarito.

## Comandos por categoria

```
DDL — estrutura
  CREATE TABLE, ALTER TABLE, DROP TABLE

DML — dados
  SELECT, INSERT, UPDATE, DELETE

DCL — controle
  GRANT, REVOKE
```
