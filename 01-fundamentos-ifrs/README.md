# 🗄️ Estudos de Banco de Dados

Repositório dedicado ao estudo de Banco de Dados, desde os conceitos fundamentais até a prática com SQL. Criado como parte da minha jornada de transição para a área de Dados.

## 👨‍💻 Sobre mim

Físico, professor e entusiasta de dados. Estudo banco de dados para complementar minha base em Python e análise de dados, com foco em me tornar Cientista de Dados.

2. Leia os conceitos em `.md` antes dos scripts `.sql`
3. Tente resolver os exercícios antes de ver o gabarito
4. Use o `cheatsheet` como referência rápida
